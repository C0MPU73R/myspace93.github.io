# Myspace Themes

Themes for http://myspace.windows93.net


## Use

Apply a theme by copying the text into the "CSS" box on your profile.


## Themes

**Facebook**

    @import url("https://myspace93.github.io/Facebook.css");

**Windows95**

    @import url("https://myspace93.github.io/Win95.css");
